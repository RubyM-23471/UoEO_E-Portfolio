Python Exercises

Python is something I have been wanting to learn more about for many years so getting the chance to dive deeper into it has been very exciting.
Below are a few exercises and questions I answered using Python during my time on the course

Question 1

Data sort using cards: Suppose you are sorting a set of cards. The right hand holds several cards which are sequentially sorted by numbers arranged from right to left, say for example, five cards with 10, 6, 5, 3, 2 arranged from right to left. Your left hand is picking up the top card from a pile of card deck. Now you insert the card that you have picked from the pile of deck into the right position within the right-hand cards, in a sorted order from right to left.
Write the instructions in a step by step manner. Note down even a minor step that you have taken to insert the left-hand card into a correct position within the set of right-hand cards so that the right-hand card set remains sorted from right to left.

For this problem, I knew I would need to compare each new card to the numbers in the list. For example,if I pulled a 7 would first comapre it to the leftmost card to see if it was smaller. If it was, I would move on to the next card and see if it was larger or smaller. If it was larger, I would place the new card before it. If it was smaller, I would move on to the next card and repeat the process.

Answering the problem with Python



import random

my_list=[10,6,5,3,2]
num=random.randint(0,10)
print(num)

for i in range(len(my_list)):
    if num < my_list[i]:
        if num >= my_list[i+1]:
            my_list.insert(i+1,num) 
            print(my_list)
